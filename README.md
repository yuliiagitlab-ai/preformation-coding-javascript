# JavaScript Fundamentals  
Preformation Coding — Bruxelles Formation  

This repository contains my JavaScript practice work completed during the **Preformation Coding program at Bruxelles Formation**.

The exercises cover core programming concepts and are organized progressively, from basic syntax to small algorithmic challenges.

---

## Topics Covered

- Variables  
- Strings  
- Numbers & Math  
- Booleans  
- Conditions  
- Arrays  
- Objects  
- Loops  
- Functions  
- Small algorithm exercises  

All practice files are located in the `JS` folder.

## How to Run

1. Open `index.html` in your browser.
2. Use the interactive selector to run different scripts.
3. Press **F12** to open the browser console and view the results.
